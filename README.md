# INTERNITY Studio

A modern, animated creative studio portfolio built with React, Vite, and Tailwind CSS.

## Tech Stack

- **React 18** with TypeScript
- **Vite** for fast builds and dev server
- **Tailwind CSS 4** for styling
- **GSAP** + **Framer Motion** for animations
- **Radix UI** + **MUI** for UI components

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Deployment

This project is configured for **Vercel** deployment. Import the repo and Vercel will auto-detect Vite settings.

- **Build Command:** `npm run build`
- **Output Directory:** `dist`
